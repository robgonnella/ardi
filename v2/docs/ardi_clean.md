## ardi clean

Delete project, or global, data directory

### Synopsis


Removes all installed platforms and libraries from project data directory. If run with "--global" all data will be removed from ~/.ardi

```
ardi clean [flags]
```

### Options

```
  -h, --help   help for clean
```
