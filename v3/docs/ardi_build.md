## ardi build

Compiles builds defined in ardi.json

### Synopsis


Compiles builds defined in ardi.json

```
ardi build [flags]
```

### Options

```
  -a, --all          Compile all builds specified in ardi.json
  -h, --help         help for build
  -s, --show-props   Show all build properties (does not compile)
```
